Car Car

An application built with a partner around a car dealership that allows users to add car manufacturers, car makes, and vehicles in the inventory using VIN. Additionally, allows for tracking of service appointments and sales.

Used a microservice approach and seperated the application into several smaller services that communicate with eachother. The service microservice, sales microservice, and inventory microservice.

Utilized Django for the backend to create various models based on the needs of microservice. Stored data using PostgreSQL. Polling was used to sync the data between our microservices. React and Bootstrap were used to create a responsive frontend.

Technologies used:

  • React JS
  • CSS
  • HTML
  • Django
  • Python
  • Docker
  • Bootstrap

Weather App

A weather application that allows users to get current weather data and a five day forecast for a location based on city and state. Once the user searches for a location, that location is saved and displayed on the frontend. Current weather data is displayed on home screen while the six day forecast for a location is displayed on the detail page for a location.

Implemented the open weather API into my backend and parsed the JSON returned to obtain and save weather data for locations that the user searches. Calculated the average max/min temperature along with average percipitatino from the JSON that was returned.

Utilized Django to make a model for locations and store the data in PostgreSQL. The frontend was designed using React, Redux and Tailwind to create a fully responsive and dynamic frontend.

Technologies used:

  • React JS
  • Redux
  • CSS
  • HTML
  • Django
  • Python
  • Tailwind
